TimeSlot

A full-stack web application that allows users to organize meetings with others at St. Francis Xavier University. The project was developed using the MERN stack, and followed the deliverable - reqiurements document structure.

Features

  • Persistant data (MongoDB Atlas)
  • Login, Register, and Logout
  • Post Meetings
  • View relavent meetings
